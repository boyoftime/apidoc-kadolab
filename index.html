Kadolab SMS API Documentation

A complete guide for developers to integrate with the Kadolab SMS API. This document covers authentication, endpoints, and provides practical Python code examples.

This documentation is also available on our GitHub Pages site.

User Registration

Create a new user account to obtain an API token. This token is essential for authenticating all subsequent API requests.

POST /register

Request Body

The following parameters must be sent in the JSON body of the request.

ParameterTypeRequiredDescription
nameStringYesThe user's full name.
emailStringYesA unique email address for the account.
passwordStringYesPassword for the account (min. 8 characters).
password_confirmationStringYesMust match the password field.
sms_gateway_usernameStringYesUsername from your SMS Gateway mobile app.
sms_gateway_passwordStringYesPassword from your SMS Gateway mobile app.
sms_gateway_urlStringYesThe gateway URL. Use the default value provided.

Python Example: Registering a User

Important: Always include the Accept: application/json header to ensure the API returns a JSON response. 
import requests
import json

# API endpoint for registration
url = "https://sms-api.kadolab.com/api/register"

# User data for registration.
# NOTE: The email must be unique for each registration attempt.
payload = {
    "name": "Jane Developer",
    "email": "jane.dev.unique@example.com",
    "password": "a-very-strong-password",
    "password_confirmation": "a-very-strong-password",
    "sms_gateway_username": "YOUR_GATEWAY_USERNAME",
    "sms_gateway_password": "YOUR_GATEWAY_PASSWORD",
    "sms_gateway_url": "https://api.sms-gate.app/3rdparty/v1/messages"
}

# Set headers to specify content type and expected response format
headers = {
    'Content-Type': 'application/json',
    'Accept': 'application/json'
}

try:
    # Send the POST request
    response = requests.post(url, headers=headers, data=json.dumps(payload))
    response.raise_for_status()  # Raise an exception for bad status codes (4xx or 5xx)

    # Process the successful response
    print("‚úÖ Registration Successful!")
    print(f"Status Code: {response.status_code}")
    
    # Pretty-print the JSON response
    api_data = response.json()
    print(json.dumps(api_data, indent=2))

except requests.exceptions.RequestException as err:
    print(f"‚ùå Request Error: {err}")

Send an SMS

Send an SMS to one or more recipients. This is a protected endpoint and requires authentication via a Bearer Token.

POST /send-sms

Authentication

Provide your API token as a Bearer Token in the Authorization header. Format: Authorization: Bearer YOUR_API_TOKEN

Request Body

ParameterTypeRequiredDescription
phoneNumbersArray of StringsYesA list of recipient phone numbers in E.164 international format (e.g., +255620123456).
messageStringYesThe text content of the SMS.

Python Example: Sending an SMS

import requests
import json

# API endpoint for sending SMS
url = "https://sms-api.kadolab.com/api/send-sms"

# Your API token obtained from the registration step
api_token = "65|abcdefgHIJKLMNOPQRSTUVWXYZ1234567890abcdef" # <-- REPLACE WITH YOUR TOKEN

# Message details
payload = {
    "phoneNumbers": ["+255695533100", "+255715123456"],
    "message": "Hello from the Kadolab API! üöÄ"
}

# Set headers, including the Authorization header for authentication
headers = {
    'Authorization': f'Bearer {api_token}',
    'Content-Type': 'application/json',
    'Accept': 'application/json'
}

try:
    # Send the POST request
    response = requests.post(url, headers=headers, data=json.dumps(payload))
    response.raise_for_status()

    # Process the successful response
    print("‚úÖ SMS Queued Successfully!")
    print(f"Status Code: {response.status_code}")
    print("Response:")
    print(json.dumps(response.json(), indent=2))

except requests.exceptions.RequestException as err:
    print(f"‚ùå Request Error: {err}")

Success Response (200 OK)

A successful request queues the SMS for delivery and returns the following status message.

{
  "status": "queued",
  "message": "SMS is being sent in the background."
}
